## 이미지 규격
- **형식**: PNG, **투명 배경**(알파). 배경색·박스 없이 캐릭터만.
- **구도**: **정면(카메라를 바라보는) 전신 1장**. 아래쪽이 발. 좌우 이동 시 게임이 자동으로 좌우 반전합니다.
- **크기**: 정사각에 가깝게. 캐릭터·몬스터 **256×256**, 보스 **512×512** 권장(고해상도로 넣으면 게임에서 자동 축소).
- **여백**: 캐릭터가 프레임을 꽉 채우되 살짝 여백. 그림자·글로우는 게임이 별도 처리하니 굽지 말 것(선택).

## 어디서 만드나 (사람/몬스터)
- **AI 이미지 생성**(권장): "정면 전신, 투명 배경, 게임 캐릭터, <설명>" 프롬프트로 생성 후 배경 제거(remove.bg 등). 예: 미드저니/DALL·E/Stable Diffusion/니지저니.
- **무료 CC0 에셋**: itch.io, OpenGameArt(CC0), Kenney — "top-down character", "monster sprite" 검색. 라이선스 확인.
